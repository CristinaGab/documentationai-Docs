<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom brand colors, like `#3B82F6`, in your documentation themes
- Introduced expandable FAQ sections for better user experience

## Improvements

- Enhanced search performance by `>30%` for large documentation sites
- Optimized image loading with lazy loading for faster page renders

## Bug Fixes

- Fixed heading hierarchy issues in generated MDX pages
- Resolved special character escaping in prose text

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated multi-language code examples with `<CodeGroup>` components
- Added `<Tabs>` for platform-specific installation guides

## Bug Fixes

- Corrected Mermaid diagram rendering as code blocks, not components
- Patched JSX attribute parsing for double quotes and proper escaping

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

## Breaking Changes

- Removed H1 headings from body content; use frontmatter titles only

## Initial Setup

- Full support for documentation workflows with realistic code examples

</Update>

  </Step>

  <Step title="GitHub Notifications" icon="github">

<Tabs>
