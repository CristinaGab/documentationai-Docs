## Brand Guidelines

Maintain a professional look with the primary brand color `#3B82F6`. Apply it to headings, links, and accents for visual unity.

<Expandable title="Advanced Brand Config" default-open="false">

Configure your theme in the settings file:

<CodeGroup tabs="JSON,YAML">
```json
{
  "theme": {
    "primaryColor": "#3B82F6",
    "headings": {
      "color": "#3B82F6"
    }
  }
}
```

```yaml
theme:
  primaryColor: "#3B82F6"
  headings:
    color: "#3B82F6"
```
</CodeGroup>

</Expandable>

## Initial Setup

Get started quickly with these steps. You complete setup in under five minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.raileanu-docs.com` with your email.
  </Step>
  <Step title="Import Project" icon="upload">
    Connect your GitHub repo or upload initial files.
  </Step>
  <Step title="Customize Theme" icon="palette">
    Set your brand color to `#3B82F6` in project settings.
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Create `introduction.mdx` with frontmatter and H2 headings.
  </Step>
</Steps>

<Tabs>
  <Tab title="CLI Setup" icon="terminal">
    Install and initialize:

    ```bash
    npm install -g raileanu-docs-cli
    raileanu-docs init my-project
    ```
  </Tab>
  <Tab title="Web UI" icon="globe">
    Use the dashboard:

    1. Log in to the web app.
    2. Select "New Space".
    3. Enter project details.
  </Tab>
</Tabs>

<Callout kind="success">
  Your docs now live at `https://docs.raileanu.com/my-project`. Share the link with your team.
</Callout>

This setup empowers you to manage complex projects efficiently while showcasing your brand identity. Explore sidebars for navigation and use search for quick access.
